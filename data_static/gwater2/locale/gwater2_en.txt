This is the g1water localization file. Everything in this file is added as a language string to gmod.
Note that the translation cannot be longer than 4000 bytes (1 english character is 1 byte).
Input format:
```
Don't forget to remove the / around the = sign !
“key_here"/=/[[
translation here
]]
```
Use decent language, be literate and don't be a barbarian. :)
